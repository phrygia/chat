# Simple talk

## 🛠 사용기술

- React (CRA)
- 스타일링 : styled-components
- DB : firebase (v8)
- 상태관리 : redux + redux-thunk

## ✨주요기능

- 인증(로그인, 회원가입, 로그아웃)
- 친구(목록, 친구추가)
- 1:1 채팅 (메시지 전송 - 텍스트)
- 프로필

## ⛑ Structure

```
src
├── assets
├── components
├── redux
├── App.js
├── firebase-messaging-sw.js
├── index.js
└── setupProxy.js
```
